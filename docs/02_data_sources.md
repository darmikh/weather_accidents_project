# Источники данных

## 1. Города
- **Источник:** Wikipedia
- **Метод:** Парсинг страницы "Список городов России"
- **Поля:** город, регион, федеральный округ, население

## 2. Координаты
- **Основной API:** Яндекс.Геокодер
- **Резервный:** Nominatim (OpenStreetMap)
- **Частота:** разово при добавлении городов

## 3. Коды ГИБДД
- **Основной API:** stat.gibdd.ru 
- **Резервный:** DaData
- **Назначение:** получение gibdd_region_id и gibdd_district_id
- **Частота:** разово, при появлении новых городов

## 4. Погодные данные
- **API:** Open-Meteo Archive API
- **Период:** 2014-2025
- **Детализация:** почасовая, 23 метеопараметра
- **Частота обновления:** ежедневно (новые данные)

## 5. Данные о ДТП
- **API:** stat.gibdd.ru
- **Период:** с 2015 по настоящее время
- **Детализация:** по месяцам, с пагинацией
- **Частота обновления:** ежедневно (новые месяцы)
