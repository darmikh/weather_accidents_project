# Дашборд в DataLens

## Структура дашборда

### Вкладка 1: Общая статистика и тренды
- Ключевые метрики (ДТП, погибшие, раненые)
- Динамика по годам
- Топ-5 типов ДТП
- Временные паттерны (часы, дни, месяцы)

### Вкладка 2: Территориальные различия
- Карта с городами (размер = кол-во ДТП, цвет = смертность)
- Таблица с относительными показателями
- Плитка с 5 гипотезами

### Вкладка 3: Погода и дорожные условия
- Смертность при разных погодных условиях
- Анализ температуры и осадков
- Влияние освещения (главный инсайт)
- Недостатки дорог

### Вкладка 4: О дашборде
- Цель исследования
- Источники данных
- Период анализа
- Объем данных
- Структура дашборда
- Дата последнего обновления
