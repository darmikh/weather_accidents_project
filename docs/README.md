## Цель проекта
Исследовать влияние погодных условий на количество и тяжесть ДТП на примере трех городов России:
- Петрозаводск
- Тамбов
- Великий Новгород

## Технологии
- Python (ETL)
- Supabase (PostgreSQL, SQL)
- DataLens (визуализация)
- GitHub Actions (автоматизация)
